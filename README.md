# Trio Admin Panel

Админ-панель для управления сотрудниками, ресторанами и справочниками.

## Технологии

- React 18 + TypeScript
- Material UI (MUI)
- React Router v6
- React Leaflet (OpenStreetMap)
- Vite

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

## Тестовые данные

- **Username**: admin
- **Password**: admin

## Структура

- `/src/api/mock` - Mock API с хранением в localStorage
- `/src/components` - Переиспользуемые компоненты
- `/src/pages` - Страницы приложения
- `/src/types` - TypeScript типы
